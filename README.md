# Executive Sales Performance & Business Insights Report

## Overview
A business analysis project focused on identifying sales trends, customer insights, and profitability using Excel with a Power BI report design.

## Business Problem
Management lacked a consolidated view of sales and profitability across regions and customer segments.

## Tools Used
- Microsoft Excel
- Power BI (Report Design)
- Pivot Tables
- Basic DAX

## Key Analysis
- Regional Sales Performance
- Product Profitability
- Customer Segment Analysis
- Monthly Sales Trends

## Outcome
Produced executive insights and business recommendations to support data-driven decision-making.
